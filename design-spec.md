# AWS GPU 实例选择器 - 销售端设计方案

## 目标用户：AWS 销售人员

### 核心需求
1. **快速响应客户需求** - 3秒内找到合适的实例推荐
2. **可视化对比** - 用图表直观展示性能差异
3. **场景导向** - 按客户业务场景快速筛选
4. **移动友好** - 支持在客户现场用手机/平板展示
5. **快速分享** - 生成链接或PDF发给客户

```

├── UI框架: Ant Design / Chakra UI (专业感强)
├── 图表库:
│   ├── Recharts (性能对比柱状图)

│   └── Radar Chart (雷达图)
├── 状态管理: Zustand (轻量级)
└── 样式: Tailwind CSS
```

## 部署建议

1. **域名**: `aws-gpu-selector.com` 或 `gpu.aws-tools.com`
2. **托管**: Vercel / AWS Amplify（自动CI/CD）
3. **CDN**: CloudFront（全球加速）
4. **Analytics**: Google Analytics + Hotjar（追踪销售使用行为）

## 示例页面流程

### 销售场景演示
1. 销售打开网站 → 选择"AI训练"
2. 系统提问"模型规模？" → 选择"大型模型 (100B-1T)"
3. 显示推荐：P6, P5en, P5
4. 点击"对比" → 看到详细对比表格
5. 点击"生成分享链接" → 复制链接发给客户
6. 客户打开链接 → 看到相同的推荐页面
7. 客户点击"联系我们" → 自动关联到该销售
