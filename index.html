  • p4d.24xlarge: 上一代旗舰，成熟稳定，预算有限时的可靠选择，配合Spot可节省70%

推理场景销售建议

  • 客户问"推理用什么"： 先问模型规模，这是关键决策因素
  • 70B以上模型： 只推荐p5en/p5e，强调高显存和HBM3e带宽优势
  • 20B-70B模型： p5是最佳选择，性能够用且成本可控
  • 小模型推理： g5即可，如果是批量推理推荐inf2省成本

快速决策指南 - 客户最常问的3个问题

Q1: "我要训练大模型，用什么？"
→ 先问模型大小
→ 100B以上：p6 或 p5en
→ 10B-100B：p5（90%客户选这个）
→ 预算有限：p4 + Spot实例
Q2: "推理部署用什么最划算？"
→ 看模型规模（关键）
→ 小模型（<20B）：g5（实时）或 inf2（批量）
→ 中型（20B-70B）：p5
→ 大模型（>70B）：p5en
Q3: "和竞品比有什么优势？"
✓ 最新硬件：B200/H200 全球首发
✓ 自研芯片：Trainium/Inferentia 性价比高30%
✓ 弹性计费：Spot实例节省70%成本
✓ 全球部署：覆盖33个区域，就近部署低延迟
