# AWS GPU 实例列表 - 需求文档

## 项目概述

本项目是一个简洁的静态网页，以表格形式展示 AWS 所有 GPU 实例类型的技术规格和基本信息，方便快速查阅和对比。

## 项目目标

- 提供清晰的 AWS GPU 实例列表
- 按发布时间排序展示实例
- 简洁的表格形式，易于快速浏览
- 纯静态页面，加载快速

## 功能需求

### 1. GPU 实例信息展示

#### 1.1 实例类型列表
以表格形式展示以下 AWS GPU 实例系列：
- **P 系列** - 通用 GPU 计算实例（训练和推理）
- **G 系列** - 图形密集型实例（渲染和推理）
- **Inf 系列** - 机器学习推理专用实例（AWS Inferentia）
- **Trn 系列** - 机器学习训练专用实例（AWS Trainium）

#### 1.2 每个实例的详细信息
表格展示以下信息（按列顺序）：
- 实例家族（如 P6、G5）
- 区域（Global 或 Global, China）
  - Global：仅在AWS全球区域可用
  - Global, China：在AWS全球区域和中国区都可用
- GPU配置（如 8卡、4卡、1卡）
- GPU型号（如 H100、A10G）
- 每GPU显存（如 80GB、24GB）
- vCPU（虚拟CPU数量）
- 系统内存（RAM大小）
- 网络带宽（如 3,200 Gbps）
- 价格(按需/us-east-1)（估算的按需小时价格，基于us-east-1区域）
- 发布时间（年份）

### 2. 排序方式
- 按发布时间排序（新到旧）
- 按实例系列分组（P系列、G系列等）

### 3. 设计风格
- 简洁专业的商务风格
- 清晰的表格布局
- 响应式设计，支持桌面和移动设备
- 易于阅读和快速浏览

> **详细的实例推荐和使用场景**：参见 [gpu.md](./gpu.md)

## 技术需求

### 1. 技术栈
- 纯静态 HTML + CSS
- 无需框架和构建工具
- 简洁的商务风格样式

### 2. 数据来源
- **主要数据源**：gpu.md 文件
  - 包含所有 GPU 实例的详细规格
  - HTML 表格内容根据 gpu.md 生成和更新
- 参考 AWS 官方文档和定价信息进行数据维护

### 3. 浏览器兼容性
- 支持现代浏览器（Chrome, Firefox, Safari, Edge）

### 4. 部署
- 适合 GitHub Pages 或任何静态托管服务

## 项目范围

### 包含的内容
- 所有 AWS GPU 实例类型的表格展示
- 按发布时间排序
- 响应式设计

### 不包含的内容
